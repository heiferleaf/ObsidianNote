| 特性       | TCP                            | UDP                                  |
| -------- | ------------------------------ | ------------------------------------ |
| **连接性**  | 面向连接                           | 无连接                                  |
| **可靠性**  | 可靠                             | 不可靠 (尽力而为)                           |
| **状态维护** | 有状态                            | 无状态                                  |
| **传输效率** | 较低                             | 较高                                   |
| **传输形式** | 面向字节流                          | 面向数据报 (报文)                           |
| **头部开销** | 20 - 60 字节                     | 8 字节                                 |
| **通信模式** | 点对点 (单播)                       | 单播、多播、广播                             |
| **常见应用** | HTTP/HTTPS, FTP, SMTP, BGP，SSH | DNS, DHCP, SNMP, TFTP, VoIP，RIP， 视频流 |
