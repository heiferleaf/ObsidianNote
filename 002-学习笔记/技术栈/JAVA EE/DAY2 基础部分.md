> **“代码是写给人看的，只是恰好能让机器执行。” —— 清晰的结构与意图，是优秀代码的基石。**

### **数据类型、输入输出与控制流**

#### **1. 基本类型与引用类型**

*   **基本数据类型**
    *   **分类**：整型、浮点型、布尔型 (`boolean`)。
    *   **运算特性**：`float` 进行 `/` 运算得到**浮点数结果**；整型进行 `/` 运算是**整除**。

*   **引用数据类型**
    *   **分类**：类、接口、数组、字符串。
    *   **赋值特性**：变量赋值是**引用传递**（复制内存地址），而非深拷贝。多个变量可指向同一对象。
    *   **字符串的不可变性**
        *   字符串是特殊的引用类型，具有 **`immutable`** 特性。
        *   任何对字符串的“修改”操作，都会**创建新的字符串对象**，而非在原有内存上更改。

*   **实践提示**：
    *   理解引用传递对于避免意外的副作用至关重要。
    *   字符串不可变使得它是线程安全的，适合作为 `Map` 的 `Key`，但频繁拼接时应使用 `StringBuilder` 以提升性能。

#### **2. 输入与输出**

*   **输出：`System.out`**
    *   它是一个 `PrintStream` 输出流。
    *   **常用方法**：
        *   `print()`: 打印不换行。
        *   `println()`: 打印并换行。
        *   `printf()`: 格式化输出。

*   **输入：`System.in`**
    *   它是一个 `InputStream` 输入流，直接使用较复杂。
    *   **通用实践**：使用 `java.util.Scanner` 进行封装，便于读取各种类型的输入。
        ```java
        Scanner scanner = new Scanner(System.in);
        String input = scanner.nextLine();
        ```

#### **3. 控制流增强**

*   **`switch` 表达式**
    *   **传统 `switch`**：每个 `case` 必须以 `break` 结束，否则会发生**穿透**。
    *   **新式 `switch` (类似模式匹配)**：
        *   使用 `case L -> ` 语法，**无需 `break`**，从根本上避免了穿透。
        *   可以作为**表达式**使用，直接返回一个值。
        *   复杂代码块可用 `{}`，并使用 `yield` 关键字返回值。
    *   **代码对比**：
        ```java
        // 传统方式 (需break)
        switch (str) {
            case "apple":
                opt = 1;
                break;
            case "peach":
                opt = 2;
                break;
            default:
                opt = 3;
        }

        // 新式表达式 (无穿透，可返回值)
        opt = switch (str) {
            case "apple" -> 1;
            case "peach" -> {
                int b = 1;
                yield b + 1; // 复杂块中用yield返回
            }
            default -> 3;
        };
        ```

*   **`for-each` 循环**
    *   用于遍历所有 `Iterable` 对象（如集合）和数组。
    *   语法简洁，避免了手动管理索引。
    *   **示例**：`for (ElementType item : collection) { ... }`

#### **4. 数组操作**

*   **遍历**
    *   **增强 `for` 循环**：`for (int x : array) { ... }`
    *   **传统 `for` 循环**：利用 `array.length` 属性。

*   **输出**
    *   直接打印数组变量输出的是其哈希码。
    *   应使用 `java.util.Arrays` 工具类的方法，如 `Arrays.toString(array)`。

*   **`Arrays` 工具库**
    *   为原生数组提供了丰富的静态方法（排序、搜索、填充、比较等），是其功能增强库。

*   **可变参数**
    *   **本质**：方法参数 `(Type... args)` 在内部被当作 `Type[]` 处理。
    *   **优势**：
        1.  调用时无需手动创建数组，传递逗号分隔的参数即可。
        2.  可以防止传入 `null`（编译器会将其视为一个空数组或 `null` 数组元素，但方法内部仍需判空）。
