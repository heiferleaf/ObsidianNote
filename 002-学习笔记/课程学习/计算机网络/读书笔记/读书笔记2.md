## 物理层的基本概念

### 2.1 物理层的基本概念

物理层考虑的是怎样才能在连接各种计算机的传输媒体上传输数据比特流，而不是某种具体的传输媒体。物理层的作用就是屏蔽掉这些通信媒体和通信手段的差异，让数据链路层感受不到这些差异。
对物理层任务的主要可以描述为确定与传输媒体的接口有关的一些特性：
1. 机械特性
2. 电气特性
3. 功能特性
4. 过程特性
具体的物理层协议种类比较多，这是因为物理连接的方式很多（比如点对点、多点连接、广播连接）、传输媒体的种类多，所以这单元最重要的还是厘清基本的概念。

### 2.2 数据通信的基础知识

#### 2.2.1 数据通信系统的模型

一个数据通信系统可以划分成三个部分：源系统（发送端、发送方）、传输系统（传输网络）、目的系统（接收端、接受方）

源系统一般包括以下部分：
- 源点（source） 源点设备生成需要传输的数据，比如计算机的键盘输入汉字，**又称为源站或者信源**。
- 发送器 通常源点生成的数字比特流要经过发送器编码之后才能在运输系统中进行传输。最经典的发送器是调制器
目的系统一般包括以下部分：
- 接收器：接受发送过来的信号，转换成目的设备能处理的信息。最经典的接收器是解调器。
- 终点（destination） 终点设备从接收器获取传输过来的数字比特流。又称为**目的站或者信宿**。
	![[数据通信系统模型.png| 数据通信系统的模型]]

源系统和目的系统之间的传输系统可以是简单的传输线，也可以是源系统和目标系统之间的网络系统。
通信的目的是传送消息，而数据是消息的实体。**信号**是数据的电气、电磁的表现。
根据信号中的数据举止方式不同，信号可分为两类：
1. 模拟信号、连续信号 - 代表信号的参数取值连续，比如调制解调器到电话端局之间的用户线上传送的是模拟信号
2. 数字信号、离散信号 - 代表信号的参数取值离散。在使用时间域的波形标识数字信号时，代表不同离散值的基本波形称为**码元**。

#### 2.2.2 有关信道的几个基本概念

**信道**一般用来表示向某一个方向传送消息的媒体。从信道的双方信息交互方式看，有三种：
1. 单向通信：又称**单工通信**，只能有一个方向通信，没有反向交互
2. 双向交替通道：又称**半双工通道**，通信双方都可以发送消息，但不能同时发送
3. 双向同时通道：又称**双工通道**，通信双方可以同时发送消息

来自**信源**的信号称为**基带信号（基本频带信号）**，像是计算机输出的文字这种信号。基带信号往往包含较多低频分量，很多信道不支持这种低频分量，那么为了解决这个问题，就要对基带信号进行调制。
调制分为两大类：
1. **基带调制**：仅仅对基带信号的波形进行变换，使其与信道特性适应，由于这种调制方式是把数字信号转换成另一种形式的数字信号，大家更喜欢把这个过程称之为**编码**
2. **带通调制**：需要使用**载波**进行调制，把基带信号的频率范围搬到较高的频段，并转换为模拟信号，更有利于在模拟信道中传输，载波后的信号称为带通信号。

**数字信号常用编码方式**：

- 不归零制：正电平代表 1 ， 负电平代表 0 
- 归零制：正脉冲代表 1 ， 负脉冲代表 0 
- 曼彻斯特编码：一个时间周期等分，周期中心一定有跳跃，向上跳代表 0 ，向下代表 1 
- 差分曼彻斯特编码：一个时间周期等分，周期中心一定有跳跃，但是看的是开始边界，有跳动代表0， 无代表1
- nB/mB 编码：用m位编码表示n位数据，有冗余的码组
- PAM4码：用4种电平编码，每一个电平编码两位
- ![[数字信号常用编码.png| 数字信号常用编码]]

**基本的调制方法：**
- 调幅（AM）：载波的振幅随着基带信号变化而变化
- 调频（FM）：载波的频率随着基带信号变化而变化
	载波频率为$f_c$,,调制后频率为$f_1,f_0$,一般要求$f_1-f_c = f_c-f_0$
- 调相（PM）：载波的初始相位随着基带信号变化而变化

#### 2.2.3 信道的极限容量

从概念上讲，限制码元在信道上的传输速率的因素有以下两个：
1. 信道能够通过的频率范围
	信道能够通过的频率范围是有限的，信号中有许多高频分量往往不能通过信道，使得码元之间的区分度不高，这叫做码间串扰。**奈奎斯特提出了奈式准则：在带宽为 W(Hz) 的低通信道中，如果不考虑噪声干扰，则码元传输的最高速率是 2W(码元/秒),速率超过这个值，就会出现严重的码间串扰**，使接收端无法识别数据

2. 信噪比
	噪声存在于所有的电子设备和通信信道中，且是随机产生的，它的瞬时值有时候会很大，使得接收端对码元的识别出错。但是当信号相对噪声比较强，噪声的影响也会相应减小，所以信噪比就很重要。下面给出信噪比的定义$$信噪比(dB) = 10 log_{10}(S/N) (dB)$$ 信号的平均功率比噪音的平均功率，用 dB 做单位
	香农提出的**香农公式**指出：信道的极限信息传输速率 C 是 $$C = W log_2(1 + S/N)$$其中 $W$ 是信道的带宽(Hz)，$S$ 是信道内信号的平均功率， $N$ 是信道内的高斯噪声功率。
	香农公式表明，信道中的带宽或是信噪比增大，信息的极限传输速率就越高。它规定了 C 的上限。

当信道的带宽和信噪比都固定时，还可以通过让每一个码元携带更多比特的信息量来提高 信息的传输速率，这也是奈奎斯特定律$$C = 2Blog_2(M)$$的由来，M是每个码元携带的bit信息。

### 2.3 物理层下的传输媒体

**传输媒体**也叫作传输媒介，它是数据传输系统中，发送器和接收器之间的物理通路。传输媒体可以分为两大类：导引型传输媒体和非导引型传输媒体。在导引型中，电磁波被导引沿着固体媒体（铜线或光纤）传播；而非导引型媒体就是自由空间，在非导引型传输媒体中电磁波的传输常被称为无线传输。

#### **2.3.1 导引型传输媒体**

1.双绞线
	双绞线是把两根互相绝缘的铜导线并排放在一起，用规则的方式绞合起来。几乎所有的电话都用双绞线来连接电话到电话交换机，而这段线也叫做用户线。
	局域网问世后，大家就开始研究怎么把传输话音信号的双绞线用来传送计算机网络中的高速数据，陆续出现了：
	**无屏蔽双绞线 UTP （Unshielded Twisted Pair）**
	**屏蔽双绞线 STP （Shielded Twisted Pair）**
	一般说的 x/UTP 是说用不同的材料对双绞线进行电磁信号的屏蔽，比如 F 代表铝箔屏蔽层， S 是金属织网屏蔽层。
	双绞线的绞合度越高，数据的传输率越高，而传输的频率越高，衰减会更大。

2. 同轴电缆
	同轴电缆由内导体铜质芯线、绝缘层、网状编织的外导体屏蔽层，以及绝缘保护套层组成。因为外导体屏蔽层，同轴电缆有很好的抗干扰能力，被用于传输较高速率的数据。
	目前同轴电缆主要用在有线电视网的居民小区中。

3. 光纤
	通信领域，信息的传输速率提高的越来越快，光纤是现在通信技术中的重要领域。
	光纤通信利用光导纤维（简称光纤）传输光脉冲进行通信，有光脉冲就是1，没有就是0，因为可见光的频率很高，所以光纤通信的带宽很大。

#### **2.3.2 非导引型传输媒体**

1. 微波
2. 无线电
3. 红外线
4. 激光

### 2.4  信道复用技术

> 常用的信道复用技术有频分复用，时分复用，统计时分复用，码分复用，和波分复用

复用是信息通信的一个基本概念：通过复用器和分用器，将原本是多路的信号用一条信道转发。

1. **频分复用（FDM）**
	将N路信号各自搬到一个频段进行传输
	使用FDM技术，让不同用户共享信道资源，就叫 FDMA 频分多址
2. **时分复用（TDM）**
	所有用户在不同的时间占用同样的频带宽度进行通信。把时间分为很多的时分复用帧（TDM帧），每一个帧等长划分时间间隔，每一路信号占据自己的一个时间间隔进行传输。
	使用TDM技术，让不同用户共享信道资源，就叫 TDMA 时分多址
3. 统计时分复用（STDM）
4. 波分复用（WDM）
5. **码分复用**（CDM）
	每一个用户可以在同样的时间使用同样的频带进行通信。
	用CDM技术，让不同用户共享通信资源，叫做 CDMA 码分多址。
	在CDMA中，每一个bit时间划分为m个码片，每一个发送数据的站点都有一个m位二进制的码片序列，不同站点的码片序列正交（内积为0），当站点发送1，就发送码片序列，发送0，就发送码片序列反码，接收站根据需要接受站点信号的码片序列，对得到的信息进行内积，就能得到发送站点的发送信息。

### 2.5 数字传输技术

最初在数字传输系统中使用的传输标准是脉冲编码调制PCM。现在的告诉数字传输系统使用同步光纤网 SONET （美国标准）或同步数字系列 （国际标准）
