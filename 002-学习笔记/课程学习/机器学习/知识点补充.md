## 流形学习

### 核心思想
**假设**: 高维数据实际嵌入在低维流形上
**目标**: 发现这个低维流形结构并实现降维
**关键洞察**: 
- 流形上邻近的样本可以用欧式距离度量
- 距离较远的样本需要沿流形表面的测地线距离(geodesic distance)度量
- 测地线距离可用邻接图上的最短路径近似

### ISOMAP (等度量映射)

#### 核心思想
保持样本间的**测地线距离**在降维前后不变。通过将测地线距离转化为邻接图上的最短路径问题来求解。

#### 算法步骤

**Step 1: 构建邻接图**
为每个样本确定其邻域，有两种方式:
- k-近邻法: 选择距离最近的k个样本
- $\epsilon$-邻域法: 选择距离小于阈值$\epsilon$的所有样本

构建邻接矩阵，其中元素定义为:
$$d_{ij} = \begin{cases} 
\|x_i - x_j\|_2 & x_j \in \text{邻域}(x_i) \\
\infty & \text{otherwise}
\end{cases}$$

**Step 2: 计算测地线距离**
使用Dijkstra最短路径算法，在邻接图上计算任意两点间的最短路径长度，得到测地线距离矩阵$D$。其中$dist_{ij}$表示样本$x_i$到$x_j$的测地线距离。

**Step 3: MDS降维**
调用MDS(多维缩放)方法，在低维空间中重构样本坐标，使得低维空间中的欧式距离尽可能接近高维空间的测地线距离。

#### 关键点
- 邻接图的构建质量直接影响测地线距离估计的准确性
- 要求流形是连通的，否则某些样本对之间距离为$\infty$
- 对邻域参数(k或$\epsilon$)的选择敏感

### LLE (局部线性嵌入)

#### 核心思想
与ISOMAP保持距离不同，LLE保持**局部线性关系**。假设每个样本可以由其邻域样本线性表出，即:
$$x_i = w_{ij}x_j + w_{ik}x_k + w_{il}x_l$$
降维后，在低维空间中保持这种线性重构关系不变。

#### 算法步骤

**Step 1: 选择邻居**
为每个样本$x_i$确定其近邻集合$Q_i$(通常用k-近邻法)。

**Step 2: 计算线性重构系数**
在高维空间中，找到最优的线性重构权重$w_{ij}$，使得每个样本能被其邻域样本最好地线性表出:
$$\min_{w} \sum_{i=1}^{m} \left\|x_i - \sum_{j \in Q_i} w_{ij}x_j\right\|^2$$
约束条件: $\sum_{j \in Q_i} w_{ij} = 1$ (保证平移不变性)

这是一个带约束的最小二乘问题，有闭式解。令$C_{jk} = (x_i - x_j)^T(x_i - x_k)$为局部协方差矩阵，则:
$$w_{ij} = \frac{\sum_{k \in Q_i} C_{jk}^{-1}}{\sum_{l,s \in Q_i} C_{ls}^{-1}}$$

**Step 3: 低维嵌入**
固定重构系数$w_{ij}$，在低维空间中求解坐标$z_i$，使得相同的线性关系继续保持:
$$\min_{z_1,...,z_m} \sum_{i=1}^{m} \left\|z_i - \sum_{j \in Q_i} w_{ij}z_j\right\|^2$$

令$Z = [z_1, z_2, ..., z_m] \in \mathbb{R}^{d' \times m}$，$W$为重构权重矩阵，构造矩阵:
$$M = (I - W)^T(I - W)$$

对$M$进行特征值分解，取**最小的$d'$个非零特征值**对应的特征向量，这些特征向量组成的矩阵即为$Z^T$。

#### 为什么这样求解?
目标函数可以改写为:
$$\min_Z \text{tr}(ZMZ^T) \quad s.t. \quad ZZ^T = I$$
这是一个带约束的迹最小化问题，通解即为$M$最小特征值对应的特征向量。

#### 关键点
- 依赖局部线性假设: 要求流形局部近似平坦
- 有闭式解，计算效率高
- 只保持局部结构，不保证全局结构

### 两种方法的本质区别

| 维度 | ISOMAP | LLE |
|------|--------|-----|
| 保持性质 | 测地线距离(全局几何) | 局部线性关系(局部几何) |
| 优化问题 | 距离保持 | 线性重构误差最小化 |
| 求解方法 | 图算法 + MDS | 最小二乘 + 特征分解 |
| 计算复杂度 | 依赖Dijkstra($O(m^2\log m)$) | 依赖特征分解($O(m^3)$) |
| 适用场景 | 流形结构清晰且连通 | 局部近似线性 |

## 神经网络

### Q1: 为什么神经网络需要激活函数（Activation Function）？

**核心**：激活函数（Activation Function）引入非线性（Non-linearity），使网络能逼近复杂函数（Approximate Complex Functions）

**如果没有激活函数（Activation Function）**：

- 无论多少层，整个网络等价于单层线性变换（Linear Transformation）
- 设两层：$y = W_2(W_1x) = (W_2W_1)x = Wx$
- 失去表达复杂非线性关系（Non-linear Relationships）的能力


### Q2: 为什么Sigmoid/Tanh容易梯度消失（Vanishing Gradient）？ReLU（Rectified Linear Unit）如何解决？

**梯度消失（Vanishing Gradient）原因**：

- Sigmoid导数 ≤ 0.25，Tanh导数 < 1
- 反向传播（Backpropagation）时多层连乘： $$\delta^{(l)} = W^{(l+1)T}\delta^{(l+1)} \odot \sigma'(net^{(l)})$$
- 多个小于1的数连乘 → 梯度指数衰减（Exponential Decay）→ 前层参数无法更新

**ReLU（Rectified Linear Unit）解决方案**：

- 正向传播区域导数恒为1，梯度不衰减
- 有效缓解梯度消失（Vanishing Gradient）
- 代价：存在神经元死亡（Dead Neuron）问题（输入 ≤ 0 时梯度为0）


### Q3: 解释BP算法（Backpropagation Algorithm）中误差信号δ（Error Signal）的含义及其反向传播机制

**误差信号（Error Signal）定义**： $$\delta_j = \frac{\partial Loss}{\partial net_j}$$

- 损失函数（Loss Function）对神经元净输入（Net Input）的梯度
- 衡量该神经元对总损失（Total Loss）的贡献

**隐藏层δ的反向传播（Backpropagation）**： $$\delta^{(l)} = W^{(l+1)T}\delta^{(l+1)} \odot \sigma'(net^{(l)})$$

**传播机制**：

1. 从输出层开始计算δ
2. 后一层δ通过权重矩阵（Weight Matrix）转置反向传播
3. 逐元素乘以当前层激活函数（Activation Function）导数
4. 逐层向前，直到输入层

> **权重更新（Weight Update）**： $$w_{ij}^{(t+1)} = w_{ij}^{(t)} - \eta \cdot \delta_{h_j} \cdot out_{h_i}$$ 其中 $\eta$ 为学习率（Learning Rate）


### Q4: 比较SGD（随机梯度下降，Stochastic Gradient Descent）、BGD（批量梯度下降，Batch Gradient Descent）和MBGD（小批量梯度下降，Mini-batch Gradient Descent）

|方法|样本数|优点|缺点|应用|
|:--|:--|:--|:--|:--|
|**BGD（批量梯度下降，Batch Gradient Descent）**|全部|梯度准确、易并行|大数据集慢|小数据集|
|**SGD（随机梯度下降，Stochastic Gradient Descent）**|1个|更新快|震荡大、不易并行|在线学习（Online Learning）|
|**MBGD（小批量梯度下降，Mini-batch Gradient Descent）**|Batch个|平衡速度与稳定性|需调Batch Size|主流|

**Batch Size选择经验**：

- 常用：32, 64, 128, 256（2的幂次）
- 太小 → 梯度噪声（Gradient Noise）大
- 太大 → 内存占用高，泛化能力（Generalization Ability）可能下降

**术语关系**： $$\text{Iterations per Epoch（每轮迭代次数）} = \frac{\text{总样本数}}{\text{Batch Size}}$$
### Q5: Adam优化器（Adam Optimizer）的优势

> **Adam（Adaptive Moment Estimation） = Momentum（动量） + RMSProp（Root Mean Square Propagation）**

**核心机制**：

- 一阶动量（First Moment）$m$：梯度（Gradient）的指数加权平均（Exponentially Weighted Average）
- 二阶动量（Second Moment）$v$：梯度平方（Squared Gradient）的指数加权平均

**相比SGD（随机梯度下降）的优势**：

1. **自适应学习率（Adaptive Learning Rate）**：每个参数独立调整
    - 梯度大的参数 → 学习率小
    - 梯度小的参数 → 学习率大
2. **平滑更新（Smooth Updates）**：考虑历史梯度，减少震荡
3. **收敛快（Fast Convergence）**：结合两种动量优点
4. **鲁棒性强（Robustness）**：对超参数（Hyperparameters）不敏感


### Q6: 防止过拟合（Overfitting）的方法

**1. Early Stopping（早停法）**（训练策略，Training Strategy）

- 监控验证集（Validation Set）误差
- 训练集误差下降但验证集误差上升时停止
- 返回验证集误差最小的模型

**2. L1/L2正则化（Regularization）**（损失函数，Loss Function）  
$$Loss_{total} = Loss + \lambda\sum w^2$$

- 限制权重（Weights）大小
- 降低模型复杂度（Model Complexity）

**3. Dropout（随机失活）**（网络结构，Network Architecture）

- 训练时随机关闭部分神经元（Neurons）
- 防止神经元间过度依赖（Co-adaptation）
- 相当于集成学习（Ensemble Learning）

**4. 标签平滑（Label Smoothing）**（数据处理，Data Processing）  
$$y_j' = (1-\epsilon)y_j + \frac{\epsilon}{K}$$

- 软化one-hot标签（One-hot Labels）
- 避免模型过于自信（Overconfidence）
- 常用 $\epsilon = 0.1 \sim 0.3$

**5. BatchNorm（批归一化，Batch Normalization）**（归一化，Normalization）

- 稳定数据分布（Data Distribution）
- 具有隐式正则化（Implicit Regularization）效果


### Q7: BatchNorm（批归一化，Batch Normalization）的作用

**操作**： $$y = \gamma \frac{x-\mu_B}{\sqrt{\sigma_B^2+\epsilon}} + \beta$$

- $\mu_B, \sigma_B$：batch统计量（Batch Statistics）
- $\gamma, \beta$：可学习参数（Learnable Parameters）

**主要作用**：

1. **稳定数据分布（Stabilize Data Distribution）** — 减少内部协变量偏移（Internal Covariate Shift）
2. **加速收敛（Accelerate Convergence）** — 降低对学习率（Learning Rate）和初始化（Initialization）的敏感度
3. **正则化（Regularization）** — 类似Dropout（随机失活）效果
4. **允许更大学习率（Larger Learning Rate）** — 梯度更稳定

**为什么加速训练**：

- 归一化使每层输入分布稳定
- 梯度更平滑
- 避免梯度消失（Vanishing Gradient）/爆炸（Exploding Gradient）


### Q8: 输出层（Output Layer）设计原则

> **目标**：使误差信号（Error Signal）形式简洁为 $\delta = \hat{y} - y$

|任务|激活函数（Activation Function）|损失函数（Loss Function）|误差信号（Error Signal）|
|:--|:--|:--|:--|
|**回归（Regression）**|线性（恒等，Identity）|MSE（均方误差，Mean Squared Error）|$\delta = \hat{y} - y$|
|**二分类（Binary Classification）**|Sigmoid|二分类交叉熵（Binary Cross-Entropy）|$\delta = \hat{y} - y$|
|**多分类（Multi-class Classification）**|Softmax|多分类交叉熵（Categorical Cross-Entropy）|$\delta_j = \hat{y}_j - y_j$|

**Softmax特殊性**：

- 每个输出都依赖所有净输入（分母耦合）
- 计算δ时需要对所有输出求和
- 不能简单分解为 $\frac{\partial Loss}{\partial \hat{y}_j} \times \frac{\partial \hat{y}_j}{\partial net_j}$


### Q9: 神经网络（Neural Network）的三大优势

**1. 非线性映射能力（Non-linear Mapping Capability）**

- 通过复杂权值（Weights）存储大量输入输出映射
- 无需显式数学表达式
- 适用场景：
    - 无法得到解析解（Analytical Solution）
    - 缺乏专家经验（Expert Knowledge）
    - 复杂模式识别任务（Complex Pattern Recognition Tasks）

**2. 容错能力（Fault Tolerance）**

- 知识分布在所有权重（Weights）中
- 个别样本的误差或噪声对整体影响小
- 允许输入样本带有较大误差

**3. 并行处理（Parallel Processing）**

- 网络结构天然支持并行计算（Parallel Computation）
- 提高训练（Training）和推理（Inference）效率
