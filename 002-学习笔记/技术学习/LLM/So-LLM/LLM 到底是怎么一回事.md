## 概念引入

LLM-大语言模型。想要搞懂这个概念，可以从语言模型开始，一种基于神经网络的深度学习范式。

对于语言模型，输入是文本序列，输出是文本序列。这牵扯到一个问题，计算机能够处理的对象、处理得到的结果都是二进制数值，换句话说，**计算机理解不了人类语言**
为了解决这个问题，我们可以将文本进行编码，最常见的像是独热编码（One-Hot Encoding）
> One-Hot Encoding : $[0,0,0,\dots,1,\dots,0]$，通过1的索引位置表示一个文本

数据的输入-计算机处理数据-计算机得到结果-结果输出，这条路走通了，接下来就是计算机如果处理的问题。

在语言模型中，常使用的是自回归算法和非自回归算法。

### 自回归算法

- 自（Auto）：自己
- 回归（Regression）：通过之前的内容预测之后的内容

可以用条件概率链形象的得到这个过程$$
p(x_{1:L}) = p(x_1) p(x_2 \mid x_1) p(x_3 \mid x_1, x_2) \cdots p(x_L \mid x_{1:L-1}) = \prod_{i=1}^L p(x_i \mid x_{1:i-1}).
$$通过前面生成的内容，决定下一步生成的内容。每一步中，按照某种算法，依据不同文本的概率，选择生成的文本。

> [!NOTE] 温度系数T（`0<=T`）
> 每一步生成词元的概率，都是通过 softmax 进行计算$$P_i = \frac{e^{z_i}}{\sum_{j=1}^{V} e^{z_j}}$$在使用温度系数的场景下，公式变形$$P_i = \frac{e^{z_i/T}}{\sum_{j=1}^{V} e^{z_j/T}}$$这意味着：
> - T 接近0：概率计算变得保守，突出放大原来较高的概率
> - T 接近无穷：概率计算很大胆，让原来较小概率放大

### 非自回归算法

和回归对应，基于输入，直接数据完整的序列。
> 这就像印刷机，不用等到前面的文本输入了，然后输出后续文本。可以并行输出

作用就是提交计算效率。
