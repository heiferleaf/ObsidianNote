## 四、简答题

22. 什么是 Java 反射机制？它解决了什么问题？  

Java 反射机制是指程序在**运行时**动态获取类的信息（字段、方法、构造器、注解等），并能对其进行操作的能力。

它解决了以下问题：

- 编译期无法确定具体类型时，运行时动态加载和调用类
- 框架需要在不依赖具体实现类的情况下，统一操作任意对象
- 支持配置化、插件化开发，降低代码耦合度
---

|对比维度|普通调用|反射调用|
|---|---|---|
|类型确定时机|编译期确定|运行时确定|
|性能|高|较低（有额外开销）|
|代码可读性|清晰直观|相对复杂|
|封装性|严格遵守访问修饰符|可突破访问限制|
|灵活性|低，固定调用|高，动态调用|
