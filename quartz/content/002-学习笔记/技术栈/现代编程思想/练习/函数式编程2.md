### 题 1

请使用 Java 函数式接口实现一个三参数加法函数的“柯里化版本”，要求：

1. 使用 `Function` 嵌套形式定义
2. 支持如下调用方式：`curriedAdd.apply(10).apply(20).apply(30)`
3. 输出结果应为 `60`

```
// 请在此处补全代码
import java.util.function.Function;

public class Main {
    public static void main(String[] args) {
        // 你的实现

        System.out.println(curriedAdd.apply(10).apply(20).apply(30)); // 期望输出: 60
    }
}
```

**答**：

```
import java.util.function.Function;

public class Main {
    public static void main(String[] args) {
        // 实现逻辑：Function<Integer, Function<Integer, Function<Integer, Integer>>>
        Function<Integer, Function<Integer, Function<Integer, Integer>>> curriedAdd = 
            a -> b -> c -> a + b + c;

        System.out.println(curriedAdd.apply(10).apply(20).apply(30)); // 输出: 60
    }
}
```

### 题 2

给定如下商品列表，请使用 Stream 管道完成数据处理，要求：

1. 过滤出价格低于 `100` 的商品
2. 对价格打九折
3. 按折后价格升序排序
4. 最终只保留商品名称与折后价格

```
import java.util.*;
import java.util.stream.*;

class Product {
    private String name;
    private double price;

    public Product(String name, double price) {
        this.name = name;
        this.price = price;
    }

    public String getName() { return name; }
    public double getPrice() { return price; }
}

public class Main {
    public static void main(String[] args) {
        List<Product> products = Arrays.asList(
            new Product("Pen", 5),
            new Product("Book", 45),
            new Product("Laptop", 999),
            new Product("Notebook", 80),
            new Product("Bag", 120),
            new Product("Ruler", 8)
        );

        // 请在此处完成处理逻辑
    }
}
```

要求最终结果形式类似：

```
[
  {name=Pen, discountedPrice=4.5},
  {name=Ruler, discountedPrice=7.2},
  {name=Book, discountedPrice=40.5},
  {name=Notebook, discountedPrice=72.0}
]
```

**答**：

```
import java.util.*;
import java.util.stream.*;

public class Main {
    public static void main(String[] args) {
        List<Product> products = Arrays.asList(
            new Product("Pen", 5),
            new Product("Book", 45),
            new Product("Laptop", 999),
            new Product("Notebook", 80),
            new Product("Bag", 120),
            new Product("Ruler", 8)
        );

        List<Map<String, Object>> result = products.stream()
            // 1. 过滤价格低于 100
            .filter(p -> p.getPrice() < 100)
            // 2. 转换为包含折后价的中间结构 (可以使用 Map 或临时记录类)
            .map(p -> {
                Map<String, Object> map = new LinkedHashMap<>();
                map.put("name", p.getName());
                map.put("discountedPrice", p.getPrice() * 0.9);
                return map;
            })
            // 3. 按折后价格升序排序
            .sorted(Comparator.comparingDouble(m -> (Double) m.get("discountedPrice")))
            // 4. 收集结果
            .collect(Collectors.toList());

        // 打印结果
        result.forEach(System.out::println);
    }
}
```
